## 附:文档如何再改

两份文档用 LaTeX 排版,源码在 `docs/tex/`。改动后重新编译:

```
export PATH="/Library/TeX/texbin:$PATH"
cd docs/tex
xelatex 设计报告.tex && xelatex 设计报告.tex   # 跑两遍生成目录页码
xelatex 过程性证明材料.tex
```

生成的 PDF 复制到 `docs/` 覆盖同名文件即为提交版。
