# 朋辈互助现状调研问卷

**这份问卷用于一项学生心理健康作品研究,匿名填写,不记名、不涉及你的隐私,数据只用于研究并接受核查。全程约 3 分钟,你可以随时退出。谢谢你的帮助!**

> 填写方法:在选中项前的 `[ ]` 里打 `√`(或把 `[ ]` 改成 `[√]`)。单选题只选一个,多选题可选多个。

---

**1. 你目前的身份是?**（单选）

- [ ] A. 普通同学
- [ ] B. 现任或曾任班级心理委员

**2. 过去一年里,有同学向你倾诉过烦恼或心事吗?**（单选）

- [ ] A. 没有
- [ ] B. 有过 1~2 次
- [ ] C. 有过 3 次及以上

**3. 当同学向你倾诉时,你最常有的感受是?**（多选）

- [ ] A. 愿意帮,但不知道该怎么回应
- [ ] B. 怕说错话、帮倒忙
- [ ] C. 想帮,但怕自己背负太重
- [ ] D. 基本能应付
- [ ] E. 有点想躲开

**4. 你回应同学时,最常做的是?**（多选）

- [ ] A. 说"想开点""别想太多"
- [ ] B. 先安静听 TA 说完
- [ ] C. 直接给建议、出主意
- [ ] D. 讲道理
- [ ] E. 转移话题、岔开
- [ ] F. 建议 TA 去找老师或家长

**5. 你是否接受过"如何回应同学倾诉"的相关训练?**（单选）

- [ ] A. 接受过系统训练
- [ ] B. 只听过讲座或看过资料
- [ ] C. 完全没有

**6. 如果同学说出"不想活了"这类话,你知道正确的处理步骤吗?**（单选）

- [ ] A. 知道,并能说出具体步骤
- [ ] B. 大概知道,但说不太清
- [ ] C. 不知道

**7. 如果有一个 AI 能模拟同学来倾诉、让你安全地练习"怎么回应",你愿意用吗?**（单选）

- [ ] A. 愿意
- [ ] B. 看情况
- [ ] C. 不愿意

**8.（仅心理委员回答）你觉得自己胜任心理委员这个岗位吗?**（单选）

- [ ] A. 胜任
- [ ] B. 勉强胜任
- [ ] C. 不太胜任,需要训练
- [ ] D. 我不是心理委员(跳过本题)
